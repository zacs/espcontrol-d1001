# Source of Truth Contract

This page is the hard ownership map for generated files.

Rules:

- Edit the authored source, not the generated output.
- If a generated output is stale, run its generator.
- If a generator changes, update this table in the same change.
- Never hand-edit generated files or generated blocks unless the generator has
  been intentionally retired.

## Broad Check

When more than one authored source changes, use the broad product check:

```bash
npm run check:product
```

For broad code, release, or generated-output changes, also run the fast local
check:

```bash
npm run check:fast
```
