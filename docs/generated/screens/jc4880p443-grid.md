The home screen uses a **3-row x 2-column** grid, giving you **6 card slots**. Any home-screen card can be turned into a [Subpage](/features/subpages) folder containing up to 5 more cards.

Flexible card sizes are supported: Single, Tall, Wide, and Large.

| Capability | Value |

| Screen | 4.3 inches, 480 x 800, Portrait |
| Processor | ESP32-P4 |
| Built-in relays | No built-in relays |
| Image-based cards | Up to 6 simultaneous Image or Media Cover Art cards |
| Rotation support | Yes |
| Browser install slug | `guition-esp32-p4-jc4880p443` |
| Ethernet option | No |
